

# תכנון: משימת למידת כתיבה ממזגת - עמוד דיגיטלי

## מה המשימה כוללת
דף עבודה עם 5 תרגילים בנושא "כתיבה ממזגת". כל תרגיל מכיל:
- טקסטים לקריאה (שני תיאורים/מאמרים)
- **חלק א': זיהוי** - שאלות קצרות (שדות טקסט)
- **חלק ב': כתיבה** - כתיבת פסקה ממזגת (textarea)

התרגילים עולים ברמת הקושי: 1-3 פשוטים, 4 בינוני, 5 מסכם עם מאמרים ארוכים.

## מבנה טכני

### 1. עמוד חדש: `src/pages/MergeWritingTask.tsx`
- עמוד עצמאי בנתיב `/merge-writing`
- מציג את 5 התרגילים בגלילה אנכית, כל תרגיל בכרטיס נפרד
- כל תרגיל כולל:
  - הוראה + טקסטים לקריאה (תוכן קבוע בקומפוננטה)
  - שדות input לשאלות הזיהוי (חלק א')
  - textarea לכתיבת הפסקה הממזגת (חלק ב') עם ספירת מילים
  - תבנית העזרה מוצגת מעל שדה הכתיבה
- הגבלת הדבקה (כמו בעמודים הקיימים)
- כפתור הגשה אחד בתחתית העמוד → מודל שם → שמירה

### 2. שמירת התשובות
- שמירה לטבלת `submissions` הקיימת
- `answer_text` יכיל JSON מובנה עם כל התשובות לכל תרגיל
- `task_id` = null (או ליצור רשומת task ייעודית)
- שדה חדש לא נדרש - נשתמש בשדות הקיימים

### 3. עמוד דשבורד: `src/pages/MergeWritingDashboard.tsx`
- נתיב `/merge-writing/dashboard`
- מציג את כל ההגשות, עם אפשרות לפתוח כל הגשה ולראות את כל התשובות מפורטות לפי תרגיל
- אפשרות מתן ציון

### 4. קישור מעמוד Admin
- הוספת כרטיס/כפתור קבוע בראש עמוד `/admin` שמקשר למשימת המיזוג
- כפתורי שיתוף (העתקת URL) ודשבורד

### 5. ניתוב (`src/App.tsx`)
- `/merge-writing` → MergeWritingTask
- `/merge-writing/dashboard` → MergeWritingDashboard

- כל העמודים הקיימים נשארים כמו שהם
